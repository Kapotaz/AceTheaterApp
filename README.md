Η παρούσα εργασία εκπονήθηκε στο πλαίσιο του μαθήματος “Αλληλεπίδραση ανθρώπου μηχανής”, στο Τμήμα Μηχανικών Πληροφορικής και Ηλεκτρονικών Συστημάτων του Διεθνές Πανεπιστημίου της Ελλάδος. 
Αντικείμενο της εργασίας αποτελεί η μελέτη, η σχεδίαση και η υλοποίηση της εφαρμογής λογισμικού με ονομασία “Ace Theater”, ενός συστήματος διαχείρισης και κράτησης εισιτηρίων θεάτρου.
